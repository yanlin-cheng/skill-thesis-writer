# 学术写作通用规范

## 一、GB/T 7714-2015 参考文献格式

### 1.1 文献类型标识

| 文献类型 | 标识代码 | 示例 |
|---------|---------|------|
| 专著 | M | 书籍、学位论文 |
| 期刊文章 | J | 学术期刊论文 |
| 会议论文 | C | 会议论文集、会议报告 |
| 报纸文章 | N | 报纸报道 |
| 报告 | R | 技术报告、研究报告 |
| 标准 | S | 国家标准、行业标准 |
| 专利 | P | 专利文献 |
| 电子文献 | EB/OL | 网页、数据库 |

### 1.2 著录格式示例

**专著**：[序号] 主要责任者. 题名[文献类型标识]. 版本项. 出版地: 出版者, 出版年: 引文页码.
```
[1] 刘国钧, 陈绍业, 王凤翥. 图书馆目录[M]. 北京: 高等教育出版社, 1957: 15-18.
```

**期刊**：[序号] 主要责任者. 题名[文献类型标识]. 刊名, 年, 卷(期): 起止页码.
```
[2] 袁训来, 陈哲, 肖书海, 等. 蓝田生物群: 一个认识多细胞生物起源和早期演化的新窗口[J]. 科学通报, 2012, 55(34): 3219.
```

**学位论文**：[序号] 主要责任者. 题名[文献类型标识]. 保存地: 保存单位, 年份: 引文页码.
```
[3] 张志祥. 间断动力系统的随机扰动及其在守恒律方程中的应用[D]. 北京: 北京大学数学学院, 1998.
```

**会议论文**：[序号] 主要责任者. 题名[文献类型标识]//编者. 文集名. 出版地: 出版者, 出版年: 引文页码.
```
[4] 钟文发. 非线性规划在可燃毒物配置中的应用[C]//赵玮. 运筹学的理论与应用: 中国运筹学会第五届大会论文集. 西安: 西安电子科技大学出版社, 1996: 468-471.
```

**电子文献**：[序号] 主要责任者. 题名[文献类型标识/文献载体标识]. (更新或修改日期)[引用日期]. 获取和访问路径.
```
[5] 萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15]. http://www.creader.com/news/20011219/200112190019.html.
```

### 1.3 引用格式细节

**正文中引用方式**：
- 顺序编码制："...的研究成果[1]..."
- 著者-出版年制："张三等（2020）的研究发现..."

**多篇引用**：[1-3] 或 [1,3,5]

**引用页码**：[1]15-18（冒号后标注页码）

## 二、论文格式规范

### 2.1 页面设置

```yaml
纸张规格: A4 (210mm × 297mm)
页边距:
  上: 2.54cm
  下: 2.54cm
  左: 3.17cm (装订侧)
  右: 3.17cm
行距: 1.25倍行距
段落: 首行缩进2字符
```

### 2.2 字体字号

```yaml
正文:
  中文: 宋体, 小四号(12pt)
  英文: Times New Roman, 12pt

标题:
  一级标题: 黑体, 三号(16pt), 居中
  二级标题: 黑体, 四号(14pt), 左对齐
  三级标题: 黑体, 小四(12pt), 左对齐

摘要:
  标题: 黑体, 小二号(18pt)
  内容: 楷体, 小四号(12pt)

参考文献:
  中文: 宋体, 五号(10.5pt)
  英文: Times New Roman, 10.5pt
```

### 2.3 图表规范

**表格规范**：
```yaml
格式: 三线表（顶线、栏目线、底线）
编号: 按章节编排，如"表3-1"、"表3-2"
位置: 表格上方居中
标题: 表号+表题，如"表3-1 描述性统计结果"
单位: 在栏目名称后标注，如"年龄(岁)"
注: 表下左对齐，以"注："开头
```

**图规范**：
```yaml
格式: JPG/PNG ≥ 300dpi，矢量图优先
编号: 按章节编排，如"图3-1"、"图3-2"
位置: 图下方居中
标题: 图号+图题，如"图3-1 研究框架图"
图例: 清晰标注，字体大小统一
```

**公式规范**：
```yaml
编号: 右对齐，如"(3-1)"
格式: 公式居中，编号在右侧
变量: 首次出现需说明物理意义
单位: 统一使用国际单位制
```

## 三、学术表达规范

### 3.1 语态使用建议

| 学科 | 被动语态比例 | 典型表达 |
|------|-------------|---------|
| 工科 | ≥60% | "实验结果表明..."、"数据被收集..." |
| 理科 | 50-70% | "可以观察到..."、"发现..." |
| 社科 | 30-50% | "本研究认为..."、"笔者认为..." |
| 人文 | 20-40% | "本文试图论证..."、"作者观察到..." |

### 3.2 连接词使用规范

**高频词汇替换表**：

| 原词 | 替代选项 |
|------|---------|
| 此外 | 无独有偶、值得注意的是、从另一维度观察、与此同时 |
| 然而 | 诚然、不可否认的是、反观之、尽管如此 |
| 因此 | 由此可见、推而广之、综合上述分析、基于上述发现 |
| 所以 | 据此可推断、基于此、上述结果表明 |
| 另外 | 从另一个角度而言、进一步分析发现、补充说明 |

### 3.3 限定词与谨慎表述

**常用限定词**：
- 一定程度上...
- 一般而言...
- 从现有数据来看...
- 在本文研究范围内...
- 初步看来...
- 可能暗示...
- 似乎表明...

**避免绝对化表述**：
- ❌ "证明了..."
- ✅ "支持了...的观点"、"表明...的可能性"

## 四、学术伦理规范

### 4.1 引用规范

**必须引用的情形**：
1. 直接引用原文（需加引号并标注页码）
2. 转述他人观点
3. 使用他人数据、图表
4. 借鉴他人研究方法

**自我引用限制**：
- 不宜过多，避免自我炒作嫌疑
- 标注与当前研究的区别与发展

### 4.2 抄袭防范

**引用比例控制**：
- 直接引用 ≤ 全文10%
- 单篇引用 ≤ 全文5%
- 连续引用 ≤ 100字

**引用格式要求**：
- 直接引用必须加引号
- 间接引用需明确标注来源
- 图表引用需获得授权

### 4.3 研究伦理（心理学/教育学/医学）

**伦理声明必备要素**：
1. 知情同意说明
2. 隐私保护措施
3. 伦理审查委员会批准号
4. 风险告知与退出权利

## 五、常见错误检查清单

### 5.1 格式错误

- [ ] 中英文标点混用
- [ ] 括号格式不统一（全角/半角）
- [ ] 数字与单位之间未空格（如"5kg"应为"5 kg"）
- [ ] 图表编号跳跃或重复
- [ ] 参考文献格式不一致

### 5.2 内容错误

- [ ] 摘要中出现引用
- [ ] 引言中直接给出结论
- [ ] 方法部分未说明样本量计算依据
- [ ] 结果部分解释性语言过多
- [ ] 讨论部分重复结果描述

### 5.3 逻辑错误

- [ ] 研究问题与方法不匹配
- [ ] 假设未在讨论中验证
- [ ] 变量定义前后不一致
- [ ] 结论超出研究范围
- [ ] 因果关系论证不足
