# GB/T 7714-2015 参考文献格式详解

## 一、著录项目与格式

### 1.1 专著（Monograph）

**标识代码**：M

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识]. 版本项. 出版地: 出版者, 出版年: 引文页码[引用日期]. 获取和访问路径.
```

**示例**：
```
[1] 刘国钧, 陈绍业, 王凤翥. 图书馆目录[M]. 北京: 高等教育出版社, 1957: 15-18.
[2] 昂温 G, 昂温 P S. 外国出版史[M]. 陈生铮, 译. 北京: 中国书籍出版社, 1988.
[3] 辛希孟. 信息技术与信息服务国际研讨会论文集: A集[C]. 北京: 中国社会科学出版社, 1994.
```

### 1.2 期刊文章（Journal Article）

**标识代码**：J

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识]. 刊名, 年, 卷(期): 起止页码.
```

**示例**：
```
[4] 袁训来, 陈哲, 肖书海, 等. 蓝田生物群: 一个认识多细胞生物起源和早期演化的新窗口[J]. 科学通报, 2012, 55(34): 3219.
[5] 李炳穆. 韩国图书馆法[J]. 图书情报工作, 2008, 56(2): 6-12.
[6] DES MARAIS D J, STRAUSS H, SUMMONS R E, et al. Carbon isotope evidence for the stepwise oxidation of the Proterozoic environment[J]. Nature, 1992, 359: 605-609.
```

**期刊页码处理**：
- 起止页码完整："15-18"
- 转页："15-18+21" 或 "15-18, 21"

### 1.3 学位论文（Dissertation）

**标识代码**：D

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识]. 保存地: 保存单位, 年份: 引文页码.
```

**示例**：
```
[7] 张志祥. 间断动力系统的随机扰动及其在守恒律方程中的应用[D]. 北京: 北京大学数学学院, 1998.
[8] CALMS R B. Infrared spectroscopic studies on solid oxygen[D]. Berkeley: Univ. of California, 1965.
```

### 1.4 会议论文（Conference Paper）

**标识代码**：C

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识]//编者. 文集名. 出版地: 出版者, 出版年: 引文页码.
```

**示例**：
```
[9] 钟文发. 非线性规划在可燃毒物配置中的应用[C]//赵玮. 运筹学的理论与应用: 中国运筹学会第五届大会论文集. 西安: 西安电子科技大学出版社, 1996: 468-471.
[10] GANZHA V G, MAYR E W, VOROZHTSOV E V, et al. Computer algebra in scientific computing: CASC 2000: proceedings of the Third Workshop on Computer Algebra in Scientific Computing[C]. Berlin: Springer, 2000.
```

### 1.5 报纸文章（Newspaper Article）

**标识代码**：N

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识]. 报纸名, 出版日期(版次).
```

**示例**：
```
[11] 丁文祥. 数字革命与竞争国际化[N]. 中国青年报, 2000-11-20(15).
[12] 张田勤. 罪犯DNA库与生命伦理学计划[N]. 大众科技报, 2000-11-12(7).
```

### 1.6 报告（Report）

**标识代码**：R

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识]. 出版地: 出版者, 出版年.
```

**示例**：
```
[13]  World Health Organization. Factors regulating the immune response: report of WHO Scientific Group[R]. Geneva: WHO, 1970.
```

### 1.7 标准（Standard）

**标识代码**：S

**著录格式**：
```
[序号] 标准编号, 标准名称[S].
```

**示例**：
```
[14] GB/T 7714-2015, 信息与文献 参考文献著录规则[S].
[15] ISO 4-1984, Documentation — Rules for the abbreviation of title words and titles of publications[S].
```

### 1.8 专利（Patent）

**标识代码**：P

**著录格式**：
```
[序号] 专利申请者或所有者. 专利题名: 专利号[文献类型标识]. 公告日期或公开日期[引用日期]. 获取和访问路径.
```

**示例**：
```
[16] 西安电子科技大学. 光折变自适应光外差探测方法: 01128777.2[P]. 2002-03-06.
[17] TACHIBANA R, SHIMIZU S, KOBAYSHI S, et al. Electronic watermarking method and system: US6915001[P]. 2005-07-05.
```

### 1.9 电子文献（Electronic Resource）

**标识代码**：EB/OL（联机网络）

**著录格式**：
```
[序号] 主要责任者. 题名[文献类型标识/文献载体标识]. (更新或修改日期)[引用日期]. 获取和访问路径.
```

**文献载体标识**：
- MT：磁带（Magnetic Tape）
- DK：磁盘（Disk）
- CD：光盘（CD-ROM）
- OL：联机网络（Online）

**示例**：
```
[18] 萧钰. 出版业信息化迈入快车道[EB/OL]. (2001-12-19)[2002-04-15]. http://www.creader.com/news/20011219/200112190019.html.
[19] Online Computer Library Center, Inc. History of OCLC[EB/OL]. [2000-01-08]. http://www.oclc.org/about/history/default.htm.
```

## 二、著录细则

### 2.1 责任者

**个人责任者**：
- 采用姓在前、名在后的著录形式
- 欧美著者的名可以用缩写字母，缩写名后省略缩写点
- 3个以内（含3个）全部列出，超过3个列前3个加"，等"或", et al"

**示例**：
```
[1] 张三, 李四, 王五. ...
[2] SMITH J, JOHNSON A B, WILLIAMS C D, et al. ...
```

**团体责任者**：
- 名称用全称

**示例**：
```
[3] 中国科学院数学研究所. ...
[4] Stanford University. Department of Computer Science. ...
```

### 2.2 题名

**一般规则**：
- 题名包括书名、刊名、报纸名、专利题名、报告名、标准名、学位论文名、档案名、舆图名、析出题名等
- 题名按文献原有文字著录

**其他题名信息**：
- 其他题名信息是指题名的副题名、说明题名文字、多卷书的分卷书名等
- 其他题名信息前用": "标识

**示例**：
```
[1] 哈利·波特与魔法石[M]. ...  # 主书名
[2] 哈利·波特与密室[M]. ...    # 系列书中的另一本
[3] 红楼梦: 脂砚斋评本[M]. ... # 副题名
```

### 2.3 版本

**第一版不著录**，其他版本说明应著录。

**示例**：
```
[1] 葛家澍, 林志军. 现代西方财务会计理论[M]. 3版. 厦门: 厦门大学出版社, 2011.  # 第三版
```

### 2.4 出版项

**出版地**：
- 出版者或发行者所在地的名称
- 同名异地或不为人们熟悉的地名，可在地名后附州名、省名、国名等

**出版者**：
- 可以按文献原形式著录，也可以按公认的简化形式或缩写形式著录

**出版日期**：
- 采用公元纪年，用阿拉伯数字著录

### 2.5 页码

**专著**：引用文献所在的页码或起止页码
**期刊**：起止页码

**示例**：
```
[1] ... 2000: 15-18.    # 起止页码
[2] ... 2000: 15.       # 单页
[3] ... 2000: 15-18+21. # 转页
```

## 三、参考文献表的组织

### 3.1 顺序编码制

**组织方式**：
- 按正文中引用文献出现的先后顺序用阿拉伯数字连续编码
- 序号置于方括号内

**正文中引用**：
- 张三等[1]发现...
- 这一结论[2-4]与...
- 文献[3-5]指出...

### 3.2 著者-出版年制

**组织方式**：
- 参考文献表按著者字顺和出版年排序
- 中文在前，西文在后

**正文中引用**：
- 张三等（2020）发现...
- 这一发现（Smith, 2019; 李四, 2020）...

## 四、常见错误与纠正

### 4.1 作者项错误

| 错误示例 | 问题 | 正确示例 |
|---------|------|---------|
| Smith John | 欧美著者名在前 | Smith J |
| 张三, 李四, 王五, 赵六 | 超过3个应加"等" | 张三, 李四, 王五, 等 |
| ZHANG SAN | 中文著者用拼音且全大写 | 张三 |

### 4.2 题名项错误

| 错误示例 | 问题 | 正确示例 |
|---------|------|---------|
| Journal of Machine Learning Research[J]. | 题名后误加文献类型标识 | Journal of Machine Learning Research. |
| 计算机学报. 2020, 43(5) | 刊名后误加句点 | 计算机学报, 2020, 43(5) |

### 4.3 出版项错误

| 错误示例 | 问题 | 正确示例 |
|---------|------|---------|
| 北京, 科学出版社 | 出版地与出版者间用逗号 | 北京: 科学出版社 |
| 2020年 | 出版年误加"年"字 | 2020 |
| vol. 45, no. 3 | 卷期格式错误 | 45(3) |

### 4.4 页码错误

| 错误示例 | 问题 | 正确示例 |
|---------|------|---------|
| pp. 15-18 | 误用页码标识 | 15-18 |
| 15--18 | 用双横线 | 15-18 |
| 15~18 | 用波浪线 | 15-18 |

## 六、学科特殊要求

### 6.1 心理学

- 期刊名使用全称（如Journal of Personality and Social Psychology，而非JPSP）
- DOI建议著录（如可用）
- 在线优先出版文献需标注"Online First"

### 6.2 计算机科学

- 会议论文占比高，注意会议名称的完整性
- 预印本（arXiv）需注明
- 技术报告需注明报告编号

### 6.3 管理学

- 经典文献需包含早期奠基性研究
- 工作论文（Working Paper）需注明来源机构
- 案例研究需注明案例来源
