# 学术统计表格生成指南

> **最后更新**：2025-04-13  
> **适用场景**：当用户需要生成符合学术规范的统计结果表格时使用此指南

---

## 核心原则

### ✅ 正确做法
- 根据用户提供的数据，按照学术规范格式化表格
- 包含必要的统计量（均值、标准差、显著性等）
- 使用标准的三线表格式（适合中文论文）或 APA 格式（英文论文）
- 提供表格标题、注释和解读建议

### ❌ 错误做法
- 不理解数据含义就机械套用模板
- 遗漏关键统计量（如效应量、置信区间）
- 格式不符合目标期刊/学校要求
- 数字精度不统一或过度/不足

---

## 第一步：确定表格类型

根据用户的统计分析需求选择合适的表格类型：

| 表格类型 | 适用分析 | 典型内容 |
|---------|---------|---------|
| **描述性统计表** | 基础数据呈现 | M, SD, 范围, 偏度, 峰度 |
| **相关矩阵** | 变量间关系 | 相关系数 r, 显著性标记 |
| **回归分析表** | 预测模型 | B, SE, β, t, p, R² |
| **方差分析表** | 组间差异 | SS, df, MS, F, p, η² |
| **中介/调节效应表** | 机制检验 | 效应值, SE, CI, 效应占比 |
| **t 检验 / 卡方表** | 两组比较 | t, df, p, Cohen's d 或 χ² |

**询问用户**：
1. 你做了什么统计分析？（描述/相关/回归/方差...）
2. 你的论文是中文学位论文还是英文期刊投稿？（格式不同）
3. 目标期刊/学校有特殊格式要求吗？

## 第二步：各类型表格的标准格式

### 2.1 描述性统计表

#### 三线表格式（中文论文常用）

```
表X  样本人口学特征及各变量描述性统计（N=XXX）

┌──────────┬─────────┬─────────┬─────────┬─────────┬────────┐
│   变量    │    M    │    SD   │  最小值  │  最大值  │  偏度  │
├──────────┼─────────┼─────────┼─────────┼─────────┼────────┤
│ 性别(男)  │   0.52  │   0.50  │   0     │   1     │ -0.04  │
│ 年龄      │  21.34  │   1.23  │  18.00  │  25.00  │  0.32  │
│ X1       │   3.45  │   0.78  │   1     │   5     │ -0.15  │
│ X2       │   4.12  │   0.65  │   2     │   5     │ -0.42  │
│ Y        │   3.89  │   0.82  │   1.50  │   5     │ -0.28  │
└──────────┴─────────┴─────────┴─────────┴─────────┴────────┘

注：M=均值；SD=标准差。分类变量报告比例。
```

**Markdown 格式示例**：

```markdown
表X  各变量的描述性统计（N=200）

| 变量 | M | SD | 最小值 | 最大值 | 偏度 | 峰度 |
|:----:|:--:|:--:|:------:|:------:|:----:|:----:|
| X1 | 3.45 | 0.78 | 1.00 | 5.00 | -0.15 | -0.32 |
| X2 | 4.12 | 0.65 | 2.00 | 5.00 | -0.42 | 0.18 |
| Y  | 3.89 | 0.82 | 1.50 | 5.00 | -0.28 | -0.11 |

注：M = 均值；SD = 标准差。N = 200。
```

#### 分组描述性统计表

```
表X  实验组与对照组的描述性统计比较

┌──────────┬──────────────┬──────────────┬──────────────┐
│   变量    │ 实验组(n=100) │ 对照组(n=100) │     t       │
│          │   (M ± SD)   │   (M ± SD)   │             │
├──────────┼──────────────┼──────────────┼──────────────┤
│ 前测成绩  │  65.23±8.45  │  64.87±9.12  │    0.29     │
│ 后测成绩  │  78.56±7.89* │  67.34±8.76  │    8.92***  │
│ 提升幅度  │  13.33±5.67* │   2.47±4.23  │   14.78***  │
└──────────┴──────────────┴──────────────┴──────────────┘

注：*p < .05, **p < .01, ***p < .001（双尾检验）
```

### 2.2 相关分析矩阵

#### Pearson / Spearman 相关矩阵

```markdown
表X  主要变量的 Pearson 相关矩阵（N=XXX）

| 变量 |   1   |   2   |   3   |   4   |   5   |
|:----:|:-----:|:-----:|:-----:|:-----:|:-----:|
| 1. X1 |   1   |       |       |       |       |
| 2. X2 | .42** |   1   |       |       |       |
| 3. X3 | .35** | .38** |   1   |       |       |
| 4. M  | .28** | .31** | .44** |   1   |       |
| 5. Y  | .41** | .36** | .39** | .52** |   1   |

注：**p < .01（双尾检验）。对角线以下显示相关系数。
```

**关键要点**：
- ✅ 只显示下三角或上三角（对角线为 1）
- ✅ 显著性标记：*p<.05, **p<.01, ***p<.001
- ✅ 注明样本量和检验方法（Pearson/Spearman）
- ✅ 如果变量较多（>6个），考虑拆分为多个小矩阵
- ❌ 不要把所有 p 值都列出来，太占空间

**显著性标记规则**：
| p 值范围 | 标记 |
|---------|------|
| p < .001 | *** |
| p < .01  | **  |
| p < .05  | *   |
| p ≥ .05  | 无标记 |

### 2.3 回归分析表

#### 多元线性回归（层次回归）

```markdown
表X  工作满意度影响因素的层次回归分析结果（N=XXX）

| 变量         | Model 1 | Model 2 | Model 3 | Model 4 |
|:------------|:-------:|:-------:|:-------:|:-------:|
| 控制变量     |         |         |         |         |
|   性别       |   .05    |   .03    |   .02    |   .01    |
|   年龄       |   .08    |   .06    |   .04    |   .03    |
|   学历       |  .12*    |   .09†   |   .07    |   .06    |
| 自变量       |         |         |         |         |
|   X1         |         |  .32***  |  .25***  |  .18**   |
|   X2         |         |  .28***  |  .21**   |  .15*    |
| 中介变量     |         |         |         |         |
|   M          |         |         |  .41***  |  .22**   |
| 调节变量     |         |         |         |         |
|   W × X1     |         |         |         |  .19**   |
|              |         |         |         |         |
| R²           |   .03    |   .21    |   .33    |   .37    |
| ΔR²         |   .03    |   .18*** |   .12*** |   .04**  |
| F            |  2.34*   | 14.56*** | 18.23*** | 17.89*** |

注：表中值为标准化回归系数（β）。† p < .10, * p < .05,
   ** p < .01, *** p < .001。
```

**关键要点**：
- ✅ 层次回归要展示 ΔR² 和 ΔF（或只标注 ΔR² 的显著性）
- ✅ 注明是标准化系数（β）还是非标准化系数（B）
- ✅ 控制变量通常放在第一层
- ✅ R² 要保留 2-3 位小数
- ⚠️ 如果调节项不显著，考虑简化模型

#### Logistic 回归（二分类因变量）

```
表X  影响因素的二项 Logistic 回归分析（N=XXX）

┌──────────┬──────────┬──────────┬──────────┬──────────┐
│  变量    │    B     │    SE    │ OR(Exp(B)│   95%CI  │
│          │          │          │    )     │          │
├──────────┼──────────┼──────────┼──────────┼──────────┤
│ 截距     │  -1.23***│   0.34   │   0.29   │[0.15,.57]│
│ X1       │   0.45** │   0.16   │   1.57   │[1.14,2.16]│
│ X2       │   0.32*  │   0.14   │   1.38   │[1.05,1.82]│
│ X3       │  -0.21   │   0.18   │   0.81   │[0.57,1.15]│
└──────────┴──────────┴──────────┴──────────┴──────────┘

注：OR = 优势比。模型拟合：χ²(3)=XX.XX, p<.001;
   Nagelkerke R² = .XX。***p<.001, **p<.01, *p<.05.
```

### 2.4 方差分析（ANOVA）表

#### 单因素 / 多因素 ANOVA

```markdown
表X  不同教学方法对学生成绩影响的单因素方差分析

| 变异来源   | 平方和(SS) | 自由度(df) | 均方(MS) |    F    |    p    |   η²   |
|:----------|:----------:|:---------:|:--------:|:-------:|:-------:|:------:|
| 组间变异   |   245.67   |     2     |  122.84  | 15.43***|  < .001 |  .136  |
│  组内变异   |  1582.34   |   198     |   7.99   |         |         │       │
│   误差     │  1582.34   │   198     │   7.99   │         │         │       │
├──────────┼──────────┼──────────┼──────────┼─────────┼─────────┼────────┤
│  总变异   │  1828.01   │   200     │          │         │         │        │
└──────────┴──────────┴──────────┴──────────┴─────────┴─────────┴────────┘

注：η² = 偏 eta 方（效应量）。***p < .001。
事后检验采用 Bonferroni 校正：
  方法A vs 方法B: p = .003; 方法A vs 方法C: p < .001;
  方法B vs 方法C: p = .128（ns）
```

**效应量参考标准（Cohen, 1988）**：

| η² 值 | 效应大小 | 解读 |
|-------|---------|------|
| .01 | 小效应 | 实际意义有限 |
| .06 | 中等效应 | 有一定实际价值 |
| .14 | 大效应 | 强大的实践意义 |

**关键要点**：
- ✅ 必须报告效应量（η² 或偏 η²）
- ✅ 多组比较时要做事后检验（post hoc），注明方法
- ✅ 注明是单因素还是多因素 ANOVA

### 2.5 中介与调节效应表

#### 中介效应（Bootstrap 法）

```markdown
表X  X 对 Y 的中介效应 Bootstrap 分析（Bootstrap = 5000）

┌─────────────┬──────────┬──────────┬────────────────────┬──────────┐
│  效应路径    │  效应值   │  标准误   │     95% CI        │ 效应占比% │
├─────────────┼──────────┼──────────┼────────────────────┼──────────┤
│ 总效应       │   0.52   │   0.07   │ [0.39, 0.67]      │  100.0%  │
│ 直接效应     │   0.28   │   0.06   │ [0.16, 0.40]      │   53.8%  │
│   间接效应   │          │          │                    │          │
│   X → M → Y │   0.24** │   0.05   │ [0.15, 0.35]      │   46.2%  │
└─────────────┴──────────┴──────────┴────────────────────┴──────────┘

注：CI = 偏差校正的百分位置信区间（不包含 0 表示显著）。
   **p < .01。中介效应占总效应的 46.2%。
```

#### 调节效应（简单斜率分析）

```markdown
表X  W 在 X 与 Y 关系间的调节效应分析

┌────────────┬──────────┬──────────┬──────────┬──────────┐
│  W 水平    │  系数 B  │    SE    │    t     │    p     │
├────────────┼──────────┼──────────┼──────────┼──────────┤
│ 低(-1SD)   │   0.12   │   0.08   │   1.50   │   .134   │
│ 中(均值)   │   0.32***│   0.06   │   5.33   │  < .001  │
│ 高(+1SD)   │   0.52***│   0.09   │   5.78   │  < .001  │
└────────────┴──────────┴──────────┴──────────┴──────────┘

注：简单斜率检验。***p < .001。
交互项 W×X: B = 0.20**, SE = 0.07, t = 2.86, p = .005
```

**关键要点**：
- ✅ 中介效应必须用 Bootstrap（推荐 5000 次抽样）
- ✅ 报告 95% 置信区间（不包含 0 = 显著）
- ✅ 调节效应配合简单斜率图更直观
- ✅ 注明调节变量在什么水平下效应显著

## 第三步：数字与格式规范

### 3.1 小数位数统一

| 统计量 | 推荐小数位 | 示例 |
|-------|----------|------|
| 均值 (M) | 2 位 | 3.45 |
| 标准差 (SD) | 2 位 | 0.78 |
| 相关系数 (r) | 2 位 | .42** |
| 回归系数 (B/β) | 2-3 位 | 0.324 或 .32** |
| 标准误 (SE) | 2-3 位 | 0.156 |
| t/F 值 | 2 位 | 5.33 |
| p 值 | 3 位（或直接用阈值） | .001 或 < .001 |
| R² / η² | 2-3 位 | .136 或 0.14 |
| OR (Logistic) | 2 位 | 1.57 |
| 置信区间 (CI) | 2 位 | [0.15, 0.57] |

### 3.2 特殊情况处理

**p 值的报告规范**：

| 情况 | 写法 | 说明 |
|------|------|------|
| p > .05 | `p = .23` 或 `.23` | 报告精确值 |
| p < .001 | `p < .001` 或 `***` | 不写 .000 |
| p = .000（软件输出） | `p < .001` | 从不写 "p = .000" |
| 边缘显著 | `p = .058` 或 `p = .06†` | 可用 † 标记 |
| 不显著 | `p > .05` 或 `ns` | ns = not significant |

**缺失值的处理**：

```markdown
✅ 正确：M = 3.45 (n = 198)，2 个缺失值采用列表删除
✅ 正确：（如有缺失，说明处理方式）
❌ 错误：忽略缺失值不说明
```

## 第四步：表格注释规范

每张表格下方都应有注释，包含以下信息：

### 必须包含的内容

```
1. 缩写解释（如果使用了缩写）
2. 显著性水平说明（*p<.05, **p<.01, ***p<.001）
3. 特殊符号说明（如 † p<.10）
4. 补充说明（如检验方法、样本量等）
```

### 注释模板

```markdown
# 中文论文通用模板
注：M = 均值；SD = 标准差。*p < .05, **p < .01, ***p < .001（双尾检验）。

# 英文 APA 格式模板
Note. M = mean; SD = standard deviation. *p < .05, **p < .01, 
      ***p < .001 (two-tailed).

# 回归分析专用
注：表中为非标准化回归系数。括号内为标准误。
      *p < .05, **p < .01, ***p < .001。

# 方差分析专用
注：SS = 平方和；df = 自由度；MS = 均方；
      η² = 偏 eta 方（Cohen, 1988）。

# 中介效应专用
Note. BCa 95% CI = bias-corrected and accelerated 95% confidence interval
      based on 5000 bootstrap samples.
```

## 第五步：质量自检清单

生成表格后，逐项检查：

```
□ 表格编号是否连续？（表1、表2、表3...）
□ 表格标题是否清晰准确？
□ 所有统计量的小数位数是否一致？
□ 是否包含了必要的统计量（如 R²、效应量）？
□ 显著性标记是否正确？
□ 注释是否完整（缩写、显著性水平、补充说明）？
□ 数字是否有异常值（如相关系数 > 1）？
□ 表格格式是否符合目标期刊/学校要求？
□ 是否需要补充图表（如简单斜率图）？
```

## 附录：常见问题解答

### Q1: 用户只给了 SPSS 输出截图怎么办？

**A**：
1. 请用户将数据导出为 Excel/CSV 格式
2. 或者让用户口述关键数值
3. 根据这些数值按上述格式重新排版
4. **不要试图从截图中 OCR 读取数字**（容易出错）

### Q2: 用户的数据看起来有问题怎么办？

**A**：
- 相关系数 > 1 或 < -1 → 数据错误，请用户核对
- 样本量 N 与自由度 df 不匹配 → 可能存在缺失值
- 所有 p 值都 < .001 → 可能是小样本过拟合或数据造假
- **主动指出可疑之处**，而不是默默接受

### Q3: 应该用三线表还是普通表格？

**A**：
| 论文类型 | 推荐格式 |
|---------|---------|
| 中文学位论文 | **三线表**（顶线、栏目线、底线） |
| 中文核心期刊 | 三线表 |
| 英文期刊（APA风格） | 三线表变体（横线分隔） |
| PPT/汇报材料 | 可以用彩色普通表格 |

### Q4: 如何处理大量变量？

**A**：
- 变量 ≤ 8：可以放一张大矩阵表
- 变量 8-15：拆成 2-3 张子表（按理论维度分组）
- 变量 > 15：只在正文中描述关键相关，其余放入附录
- **不要强行塞进一张超宽的表**

---

> 💡 **使用提示**：统计表格的核心价值在于**清晰传达数据信息**。AI 应当帮助用户整理数据、规范格式，但最终的统计结果必须基于用户提供的真实分析。遇到不确定的统计细节，应当向用户确认而非猜测。
