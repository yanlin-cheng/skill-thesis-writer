# GB/T 7714-2015 参考文献格式化指南

> **最后更新**：2025-04-13  
> **适用标准**：GB/T 7714-2015《信息与文献 参考文献著录规则》  
> **适用场景**：当用户需要格式化参考文献、检查引用规范时使用此指南

---

## 核心原则

### ✅ 正确做法
- 逐条检查每条文献的完整性和规范性
- 根据文献类型选择正确的格式模板
- 智能识别并修正常见错误
- 提供清晰的修改建议和对比

### ❌ 错误做法
- 使用正则表达式机械解析（容易误判）
- 盲目相信用户输入的原始信息（可能有错）
- 只改不改，不解释原因
- 忽略学科特殊要求

---

## 第一步：识别文献类型

GB/T 7714-2015 规定的主要文献类型及标识代码：

| 文献类型 | 类型标识 | 常见示例 |
|---------|---------|---------|
| 期刊论文 | [J] | 学术期刊发表的文章 |
| 专著（图书） | [M] | 教材、专著、译著 |
| 学位论文 | [D] | 硕士/博士学位论文 |
| 会议论文集 | [C] | 会议论文集中的文章 |
| 报告 | [R] | 技术报告、研究报告 |
| 标准 | [S] | 国家标准、行业标准 |
| 专利 | [P] | 发明专利、实用新型 |
| 报纸文章 | [N] | 报纸发表的学术文章 |
| 电子资源 | [EB/OL] | 网页、在线数据库 |
| 数据库 | [DB/OL] | 在线数据库条目 |
| 计算机程序 | [CP] | 软件、程序 |

**⚠️ 常见混淆**：
- 论文集中的文章用 **[C]**，不是 **[J]**
- 网络获取的电子期刊仍用原类型标识（如 **[J]**），但需补充引用日期和 URL
- 学位论文是 **[D]**，不是 **[J]**

## 第二步：各类型文献的标准格式

### 2.1 期刊论文 [J]

#### 标准格式

```
[序号] 作者. 题名[J]. 刊名, 出版年, 卷(期): 起止页码.
```

#### 完整示例

**中文期刊**：
```
[1] 张三, 李四, 王五. 深度学习在图像识别中的应用研究[J]. 计算机学报, 2023, 46(5): 1023-1035.
```

**英文期刊**：
```
[2] Smith J, Johnson A, Williams B. Deep learning for image recognition[J]. IEEE Transactions on Pattern Analysis and Machine Intelligence, 2023, 45(3): 567-589.
```

#### 关键要点

- ✅ 作者之间用逗号分隔，最后两位作者间用逗号 + "等"（中文）或 "et al"（英文）
- ✅ 超过 3 位作者时，列出前 3 位后加 "，等" 或 "，et al"
- ✅ 卷号和期号格式：`卷(期)` 或 `卷(期): 页码`
- ✅ 页码用起止页码，如 `1023-1035`
- ❌ 不要遗漏卷号或期号（除非该刊没有）

---

### 2.2 专著（图书）[M]

#### 标准格式

```
[序号] 作者. 书名[M]. 版本(第1版不标注). 出版地: 出版者, 出版年: 引用页码.
```

#### 完整示例

**普通专著**：
```
[3] 李明. 机器学习导论[M]. 北京: 清华大学出版社, 2022: 156-178.
```

**译著**：
```
[4] Goodfellow I, Bengio Y, Courville A. 深度学习[M]. 赵申剑, 等译. 北京: 人民邮电出版社, 2017.
```

**多卷书**：
```
[5] 王教授. 中国教育史纲[M]. 第2卷. 上海: 上海教育出版社, 2020.
```

#### 关键要点

- ✅ 第 1 版不标注版本号
- ✅ 出版地要具体到城市名
- ✅ 引用特定页码时需注明（可选）
- ❌ 不要把出版社所在省/州也写上（除非必要区分）

---

### 2.3 学位论文 [D]

#### 标准格式

```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.
```

#### 完整示例

**硕士学位论文**：
```
[6] 刘洋. 基于Transformer的文本分类方法研究[D]. 北京: 北京大学, 2023.
```

**博士学位论文**：
```
[7] Chen W. Research on natural language processing[D]. Stanford: Stanford University, 2022.
```

#### 关键要点

- ✅ 保存地指学位授予单位所在城市
- ✅ 保存单位指大学或研究院全称
- ❌ 不要写导师姓名（GB/T 7714 不要求）

---

### 2.4 会议论文 [C]

#### 标准格式（论文集中析出的文献）

```
[序号] 作者. 题名[C]//编者. 论文集名. 出版地: 出版者, 出版年: 页码.
```

#### 完整示例

**会议论文集**：
```
[8] 赵六, 孙七. 一种新的目标检测算法[C]//李八, 主编. 第十届全国计算机视觉大会论文集. 北京: 科学出版社, 2023: 234-240.
```

**国际会议**：
```
[9] Brown K, Davis L. Neural architecture search[C]//Proceedings of the 38th International Conference on Machine Learning. Virtual: PMLR, 2021: 1234-1245.
```

#### 关键要点

- ✅ 使用 `//` 分隔论文信息和论文集信息
- ✅ 编者后可加 "主编" 或 "ed." / "eds."
- ✅ 国际会议常用 "Proceedings of..." 作为论文集名
- ⚠️ 有些学校要求会议论文按 [J] 格式著录，请以学校要求为准

---

### 2.5 电子资源 [EB/OL]

#### 标准格式

```
[序号] 作者. 题名[EB/OL]. (发布日期)[引用日期]. URL.
```

#### 完整示例

**网页文章**：
```
[10] OpenAI. GPT-4 technical report[EB/OL]. (2023-03-27)[2023-10-15]. https://arxiv.org/abs/2303.08774.
```

**在线数据库**：
```
[11] 国家统计局. 2023年国民经济和社会发展统计公报[EB/OL]. (2024-02-29)[2024-03-10]. http://www.stats.gov.cn/tjsj/zxfb/202402/t20240228_1947915.html.
```

#### 关键要点

- ✅ 发布日期和引用日期都要有
- ✅ URL 要完整可访问（测试一下！）
- ✅ 如果是网络获取的期刊/图书，保留原类型标识 + 补充电子信息

**混合格式示例**（纸质期刊的网络版）：
```
[12] 作者. 题名[J/OL]. 刊名, 年, 卷(期): 页码(引用日期). URL.
```

## 第三步：作者字段规范化

### 3.1 中文作者

**格式规则**：
- 姓前名后（中文习惯）：张三、李四
- 多位作者：用逗号分隔
- 超过3位：前3位 + "，等"

**示例**：
```
✅ 正确：张三, 李四, 王五, 等
✅ 正确：张三, 李四
❌ 错误：张三; 李四; 王五（分号不对）
❌ 错误：张三、李四、王五（顿号不对）
```

### 3.2 英文作者

**格式规则**：
- 姓的全称 + 名的首字母（大写，加点或不加）
- 多位作者：用逗号分隔
- 超过3位：前3位 + ", et al"

**示例**：
```
✅ 正确：Smith J A, Johnson B C, et al
✅ 正确：Smith JA, Johnson BC, et al
✅ 正确：Smith, J.A., Johnson, B.C., et al.（不同学校要求可能不同）
❌ 错误：John Smith（名在前了）
❌ 错误：J. A. Smith & B. C. Johnson（& 符号不对）
```

### 3.3 特殊情况

**团体作者**：
```
✅ 正确：教育部高等教育司
✅ 正确：World Health Organization
✅ 正确：IEEE Computer Society
```

**匿名作者**：
```
✅ 正确：佚名（如果确实无法确定作者）
✅ 更好：尽量找到真实作者信息
```

## 第四步：常见错误检测与修正

### 4.1 缺失项检测

逐项检查清单：

```
□ 序号是否连续？从[1]开始？
□ 作者是否齐全？（至少要有第一作者）
□ 题名是否完整？（不要缩写）
□ 文献类型标识是否正确？[J]/[M]/[D]/[C]
□ 期刊/出版者名称是否为官方全称？
□ 年份是否正确？（不是出版年就是上线年）
□ 卷号、期号是否有？（期刊必须有）
□ 页码是否完整？（起止页码）
□ 对于电子资源：URL 是否有效？
□ 对于电子资源：引用日期是否有？
```

### 4.2 常见错误对照表

| 错误类型 | 错误示例 | 正确写法 | 说明 |
|---------|---------|---------|------|
| **缺类型标识** | 张三. 文章标题. 期刊, 2023 | 张三. 文章标题[J]. 期刊, 2023 | 必须加 [J] |
| **缺卷期** | ...计算机学报, 2023: 1023 | ...计算机学报, 2023, 46(5): 1023 | 期刊必须标明卷(期) |
| **作者过多** | 张三, 李四, 王五, 赵六, 孙七... | 张三, 李四, 王五, 等 | 超3位加"等" |
| **页码格式** | : p1023-p1035 | : 1023-1035 | 不要加p/P |
| **英文大小写** | smith j, johnson b | Smith J, Johnson B | 姓首字母大写 |
| **出版地过细** | 北京市海淀区清华园 | 北京 | 只写到城市 |
| **URL无效** | http://example.com/article（404） | 测试有效的URL | 务必验证 |
| **电子资源缺日期** | ...[EB/OL]. URL | ...(2023-10-01)[2023-11-01]. URL | 需两个日期 |

## 第五步：排序规则

### 5.1 顺序编码制（最常用）

按文献在正文中出现的顺序编号，即：

```
[1] 第一篇引用的文献...
[2] 第二篇引用的文献...
[3] 第三篇引用的文献...
```

**要求**：
- 编号连续，不能跳号
- 正文中的上标编号要与列表对应
- 同一文献多次引用，使用同一编号

### 5.2 作者-年份制（部分社科期刊使用）

```
(Tom, 2020)
(Smith & Johnson, 2021)
(张三, 2022; 李四, 2023)
```

**排序方式**：
- 先按第一作者姓氏字母/拼音排序
- 同一作者按年份排序
- 同一作者同年份按题名排序

## 第六步：质量检查与优化建议

### 6.1 去重检测

**检测逻辑**（人工判断）：
1. **完全重复**：同一文献出现两次 → 删除一条
2. **近似重复**：同一篇文章的不同版本（如预印本和正式版）→ 保留正式版
3. **版本选择优先级**：正式出版 > DOI 版本 > arXiv > 网页

**示例**：

❌ **重复**：
```
[1] Smith J. Paper title[J]. Journal A, 2023, 45(2): 100-110.
[5] Smith J. Paper title[J]. Journal A, 2023, 45(2): 100-110.
```

✅ **修正**：删除 [5]，后续编号顺延

---

### 6.2 时效性分析

**检查指标**：
- 近 5 年文献占比（不同学科要求不同）
- 经典文献（奠基性工作）是否包含
- 是否过度依赖老旧文献

**学科时效性要求参考**：

| 学科领域 | 建议近5年占比 | 说明 |
|---------|-------------|------|
| **工科（计算机/AI）** | ≥ 50% | 技术迭代快 |
| **心理学** | ≥ 35% | 经典理论重要 |
| **教育学** | ≥ 30% | 兼顾经典与新知 |
| **管理学** | ≥ 30% | 平衡理论与实践 |

**输出格式**：
```
📊 文献时效性分析报告
━━━━━━━━━━━━━━━━━━━━━
总文献数：85 篇
近5年文献：38 篇 (44.7%)
年份范围：2018 - 2024
最旧文献：Piaget (1952) ← 经典理论
最新文献：Zhang et al. (2024)

✅ 时效性良好（工科建议≥40%）

⚠️ 注意：
- 有 3 篇 2000 年前的文献，确认是否为经典引用
- 建议补充 2023-2024 年的最新综述
```

---

### 6.3 合规性检查

根据学科要求进行专项检查：

**通用检查项**：
```
□ 所有文献类型标识正确
□ 作者格式统一（中文/英文分开处理）
□ 期刊名使用官方全称（非缩写）
□ 页码完整且格式统一
□ 无缺失关键信息项
□ 排序符合要求（顺序编码制）
□ 无重复文献
□ URL 可访问（针对电子资源）
```

**工科专项**：
```
□ 会议论文是否标注 [C] 而非 [J]
□ 是否包含足够的近期顶会论文（CVPR/ICCV/NeurIPS 等）
□ 技术报告/标准是否正确标注 [R]/[S]
```

**社科专项**：
```
□ 量表类文献是否注明出处和修订情况
□ 经典理论文献是否包含原始文献
□ 中文文献和英文文献比例是否合理
```
