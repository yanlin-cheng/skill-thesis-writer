# 心理学类论文写作指南

## 一、心理学论文结构规范

### 1.1 APA格式基本结构

```
标题页
    - 论文标题（简洁，≤12词）
    - 作者信息
    - 作者单位
    - 简短标题（Running head）

摘要（Abstract，150-250字）
    - 研究目的（1句）
    - 方法简述（2-3句）
    - 主要结果（2-3句）
    - 结论/意义（1-2句）
    - 关键词（3-5个）

正文
    1 引言（Introduction）
        1.1 研究背景
        1.2 文献综述
        1.3 研究缺口
        1.4 研究目的与假设
    
    2 方法（Method）
        2.1 被试（Participants）
        2.2 研究设计
        2.3 研究材料与工具
        2.4 研究程序
        2.5 数据分析方法
    
    3 结果（Results）
        3.1 描述性统计
        3.2 假设检验结果
        3.3 补充分析
    
    4 讨论（Discussion）
        4.1 主要发现总结
        4.2 与已有研究的比较
        4.3 理论贡献
        4.4 实践意义
        4.5 局限性与未来研究
    
    5 结论（Conclusion）

参考文献（References）

附录（Appendix）
    - 完整量表
    - 伦理审批文件
    - 补充材料
```

### 1.2 各部分页数建议

| 部分 | 硕士论文 | 博士论文 | 期刊论文 |
|------|---------|---------|---------|
| 引言 | 8-15页 | 15-30页 | 2-4页 |
| 方法 | 5-8页 | 8-15页 | 1-2页 |
| 结果 | 8-15页 | 15-25页 | 1-3页 |
| 讨论 | 10-15页 | 20-30页 | 2-4页 |

## 二、方法部分详细规范

### 2.1 被试信息报告模板

```markdown
## 被试信息

本研究采用方便抽样/分层随机抽样，共招募被试{N}名。
其中，男性{XX}名（{XX}%），女性{XX}名（{XX}%）。
年龄范围为{XX}-{XX}岁（M = {XX}, SD = {X.XX}）。
被试主要来自{某地区/某高校}，专业背景包括{列举主要专业}。

### 纳入标准
- 年龄18-35岁
- 视力/矫正视力正常
- 无精神疾病史
- 近三个月未参与类似研究

### 样本量确定
基于G*Power 3.1进行功效分析，设定效应量f = 0.25（中等效应），
显著性水平α = 0.05，统计功效1-β = 0.80，计算所需样本量为{N}。
考虑{XX}%的流失率，最终招募{N}名被试。

### 伦理审批
本研究获{某大学}伦理委员会批准（审批号：XXX-2024-XXX）。
所有被试在实验前签署知情同意书，并告知可随时退出研究。
实验结束后，每位被试获得{XX}元报酬/课程学分。
```

### 2.2 量表报告规范

**量表使用报告模板**：

```markdown
## 测量工具

### 1. XXX量表
采用{作者}（{年份}）编制的XXX量表测量{某构念}。
该量表共{XX}个题项，采用Likert {X}点计分（1 = 完全不同意，5 = 完全同意）。
样题包括："{题项内容}"、"{题项内容}"。
在本研究中，该量表的Cronbach's α系数为{0.XX}，表明信度良好。
验证性因子分析结果显示，模型拟合指标为：
χ²/df = {X.XX}, CFI = {0.XX}, TLI = {0.XX}, RMSEA = {0.XX}（90% CI: {0.XX}-{0.XX}），
SRMR = {0.XX}，表明结构效度可接受。

### 2. XXX量表（若为翻译/修订量表）
本研究采用{某量表}的中文修订版（{修订者}，{年份}）。
原量表由{原作者}（{年份}）编制，包含{XX}个维度。
{修订者}（{年份}）在{某群体}中进行修订，报告Cronbach's α为{0.XX}-{0.XX}。
本研究中，该量表的Cronbach's α系数为{0.XX}。
```

### 2.3 研究设计报告

**实验设计报告模板**：

```markdown
## 研究设计

本研究采用{X}（组间/组内/混合）实验设计。
自变量为{XXX}（{水平数}个水平：{水平A}、{水平B}...），
因变量为{XXX}，通过{某指标}测量。
控制变量包括{列举主要控制变量}。

### 实验流程
1. 被试到达实验室后，主试介绍实验流程并获取知情同意。
2. 被试完成{某前测}。
3. 随机分配至{某条件}，进行{某操纵/干预}。
4. 完成后测测量{某变量}。
5. 填写人口学问卷和操纵检验题项。
6. 实验结束，发放报酬并感谢参与。

整个实验历时约{XX}分钟。
```

## 三、统计结果报告规范

### 3.1 描述性统计

**报告模板**：

```markdown
## 描述性统计

表1 各变量的描述性统计与相关矩阵

| 变量 | M | SD | 1 | 2 | 3 |
|------|-----|-----|-----|-----|-----|
| 1. 变量A | 3.52 | 0.84 | 1 | | |
| 2. 变量B | 2.98 | 0.76 | .42** | 1 | |
| 3. 变量C | 3.15 | 0.91 | .35** | .28* | 1 |

注：*p < .05, **p < .01, ***p < .001。对角线为AVE平方根。
```

### 3.2 假设检验报告

**t检验报告**：
```
独立样本t检验结果显示，实验组（M = 3.85, SD = 0.72）
与对照组（M = 3.12, SD = 0.68）在{某变量}上存在显著差异，
t({自由度}) = {t值}, p = {p值} < .001, Cohen's d = {效应量}（大效应）。
```

**方差分析报告**：
```
2×2混合设计方差分析结果显示：
{某因素}的主效应显著，F({df1}, {df2}) = {F值}, p = {p值} < .001, 
偏η² = {效应量}（大效应）；
{另一因素}的主效应不显著，F({df1}, {df2}) = {F值}, p = {p值} > .05；
交互作用显著，F({df1}, {df2}) = {F值}, p = {p值} < .01。
简单效应分析显示...（进一步解释）
```

**回归分析报告**：
```
层次回归分析结果如表X所示。在控制{控制变量}后，
{自变量}显著预测{因变量}，β = {标准化系数}, 
t = {t值}, p = {p值} < .001，解释额外{X}%的变异（ΔR² = {XX}）。
```

**中介效应报告**：
```
采用Bootstrap法（抽样5000次）检验中介效应。
结果显示，{自变量}对{因变量}的总效应显著（β = {XX}, p < .001）。
在纳入{中介变量}后，{自变量}的直接效应降低至β = {XX}（p = {XX}），
但仍显著/不再显著。中介效应量为{XX}，95% CI: [{XX}, {XX}]，不包含0，
表明{中介变量}的中介效应显著。
```

### 3.3 效应量报告

**常用效应量标准**：

| 效应量指标 | 小效应 | 中等效应 | 大效应 |
|-----------|-------|---------|-------|
| Cohen's d | 0.20 | 0.50 | 0.80 |
| η²/偏η² | 0.01 | 0.06 | 0.14 |
| R² | 0.02 | 0.13 | 0.26 |
| f² | 0.02 | 0.15 | 0.35 |

**报告规范**：
- 必须报告效应量，不能仅报告p值
- 在首次报告效应量时说明大小标准
- 结合效应量和样本量解释实际意义

## 四、伦理规范

### 4.1 伦理声明必备要素

```markdown
## 伦理声明

本研究遵循{某大学}心理学研究伦理规范，并获得伦理委员会批准
（审批号：XXX-2024-XXX）。具体措施包括：

1. **知情同意**：所有被试在实验前签署知情同意书，
   明确告知研究目的、流程、潜在风险与收益、保密措施及退出权利。

2. **隐私保护**：所有数据匿名编码处理，可识别信息单独存储，
   仅研究团队可访问。数据保存期限为研究结束后{X}年。

3. **风险最小化**：本研究不涉及身体伤害或严重心理创伤风险。
   如被试在实验过程中感到不适，可随时无条件退出并获得全额报酬。

4. **事后说明**：实验结束后，主试向被试说明研究真实目的，
   并回答被试疑问。如涉及欺骗，需进行充分解释。

5. **数据使用**：研究数据仅用于学术发表，不用于商业目的。
   被试有权要求删除其数据。
```

### 4.2 特殊人群研究伦理

**未成年人研究**：
- 必须获得监护人书面同意
- 获得未成年人本人知情同意（口头或简化版书面）
- 学校/机构许可

**临床人群研究**：
- 医生/治疗师转介
- 排除严重精神障碍
- 提供转介资源信息

## 五、写作要点

### 5.1 语态与句式

**推荐语态比例**：
- 主动语态：50-60%
- 被动语态：40-50%

**常用表达**：
- "本研究发现..."
- "我们观察到..."
- "这一结果与{某理论}的预测一致..."
- "一个可能的解释是..."

### 5.2 谨慎表述

**避免绝对化**：
- ❌ "证明了..."
- ✅ "支持了...的观点"
- ❌ "表明了因果关系..."
- ✅ "提示了...的可能性"

**常用限定词**：
- "初步看来..."
- "一定程度上..."
- "在本文研究范围内..."
- "可能暗示..."

### 5.3 讨论部分写作

**讨论结构**：
1. **主要发现重述**（不要重复数据，而是解释意义）
2. **与已有研究比较**（一致性+差异性）
3. **理论贡献**（支持/修正/扩展了哪些理论）
4. **实践意义**（对教育/临床/组织的启示）
5. **局限性与未来研究**

**比较研究模板**：
```
本研究发现{某结果}，这与{作者}（{年份}）的研究结论一致，
支持了{某理论}的观点。然而，我们的效应量（d = {XX}）
小于{作者}报告的效应量（d = {XX}），这种差异可能源于
{某因素}。具体而言，本研究使用{某方法}，而{作者}使用{另一方法}，
这提示{某启示}。
```

## 六、常见陷阱与规避

### 6.1 方法部分陷阱

| 陷阱 | 问题 | 规避方法 |
|------|------|---------|
| 量表来源不明 | 信效度存疑 | 明确报告量表来源、修订情况 |
| 信度报告缺失 | 数据质量未知 | 报告所有量表的Cronbach's α |
| 样本量无依据 | 统计功效不足 | G*Power功效分析 |
| 随机化程序不明 | 实验严谨性不足 | 详细说明随机化方法 |

### 6.2 结果部分陷阱

| 陷阱 | 问题 | 规避方法 |
|------|------|---------|
| 只报告p值 | 信息不完整 | 同时报告效应量、置信区间 |
| 多重比较不校正 | 假阳性风险 | Bonferroni/FDR校正 |
| 事后分析不说明 | 探索性伪装验证性 | 明确区分先验/事后假设 |
| 异常值不处理 | 影响结果稳健性 | 报告异常值检测与处理 |

### 6.3 讨论部分陷阱

| 陷阱 | 问题 | 规避方法 |
|------|------|---------|
| 重复结果 | 冗余 | 重在解释而非重复 |
| 过度推广 | 外部效度问题 | 限定结论适用范围 |
| 忽视零结果 | 选择性报告 | 如实报告所有假设检验 |
| 局限性程式化 | 显得不够真诚 | 具体说明每个局限的影响 |

## 七、推荐工具

- **统计软件**：SPSS, R, JASP, jamovi
- **功效分析**：G*Power
- **中介分析**：PROCESS (Hayes), Mplus
- **结构方程**：AMOS, Mplus, lavaan (R)
- **文献管理**：Zotero, EndNote
- **写作格式**：APA 7th版手册
